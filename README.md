# Local Development

Clone the repository

```bash
git clone https://github.com/Amirjon06/portfolio.git
```

Enter the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open

```
http://localhost:3000
```
